Fake News Detection in Country

Machine Learning Meets Fake News


From Stats to Smart Models

In a world flooded with digital content, detecting fake news isn’t just a skill — it’s a necessity. This project goes beyond basic stats and dives into the world of machine learning to classify misinformation with precision. By exploring datasets like Bangla News, LIAR, and IFND, we trained models to recognize the red flags of misinformation.

The journey includes data cleaning, visual analysis, and predictive modeling using algorithms like Naive Bayes, SVM, Random Forest, and more. The goal? Build systems that don't just read — they understand and judge the credibility of news.

Machine Learning Modules


TF-IDF Vectorization

Text data was transformed into numerical vectors using TF-IDF, capturing the importance of words across documents.


Model Training & Evaluation

Supervised learning models were trained using a 70-30 train-test split to classify news as fake or real.


Naive Bayes Classifier

A probabilistic model known for fast computation; suitable for high-dimensional text classification tasks.


Logistic Regression

A linear model used to predict binary outcomes. Achieved high accuracy in detecting fake news.


Random Forest Classifier

An ensemble model using multiple decision trees for robust classification and reduced overfitting.


Support Vector Machine (SVM)

A margin-based classifier effective in high-dimensional spaces, offering a trade-off between precision and recall.


K-Nearest Neighbors (KNN)

A distance-based classifier which compares feature vectors to neighbors for classification.


Decision Tree Classifier

A rule-based model which splits data based on feature importance for classification decisions.


Model Performance Metrics

Confusion matrices, precision-recall curves, and accuracy scores were used to evaluate each model's performance.




Logistic Regression


High Accuracy Linear Classifier

Logistic Regression was trained on TF-IDF features to distinguish between fake and real news. It performed with high accuracy and balanced precision-recall metrics, making it one of the most reliable models in the study.

The confusion matrix shows a good balance of true positives and true negatives, while the precision-recall curve highlights strong recall capability with minimal trade-off in precision.



Random Forest Classifier


Robust Ensemble Model

The Random Forest model combines multiple decision trees to improve classification performance and reduce overfitting. It was one of the top-performing models in accuracy and consistency across metrics.

It performed well across both classes, as seen in the confusion matrix. The precision-recall curve also reflects its balanced prediction strength.



Naive Bayes


Efficient Probabilistic Model

Naive Bayes offers fast classification using probabilistic inference. Although slightly lower in accuracy than other models, it remains an efficient choice for large-scale and real-time fake news detection.

Precision-recall analysis shows trade-offs, but the confusion matrix indicates reliable detection of both fake and real classes in a short time frame.



K-Nearest Neighbors (KNN)


Instance-Based Learning

KNN classifies articles based on the labels of nearby examples in the feature space. It is simple and interpretable but can be computationally expensive on large datasets.

The model shows average performance, with slightly more false positives. The precision-recall curve reflects its dependency on proper value of ‘K’ and distance metrics.



Decision Tree Classifier


Rule-Based Learning

Decision Tree builds a model of decisions based on feature importance. It’s highly interpretable but can be prone to overfitting if not properly pruned.

While the confusion matrix shows reasonable classification capability, the precision-recall curve highlights areas where the model could misclassify under uncertainty.
